# History Docs

작업 기록, 리뷰 결과, 검증 로그를 보관한다.

| 경로 | 역할 |
| --- | --- |
| `rebuild-results/` | 구현/수정 단위별 작업 결과와 검증 기록 |

`history` 문서는 현재 기준의 source of truth가 아니다. 현재 동작 기준은 `docs/stable/`에서 확인한다.
