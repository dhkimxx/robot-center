# Roadmap

## Next

- Browser 화면에서 mission 종료/재시작 흐름을 직접 스크린샷으로 검증한다.
- Robot팀과 `missionId`, `roomId`, `robotCode`, `streaming status` 계약을 확인한다.
- 과거 삭제된 작업 로그를 복구할지, git history에만 둘지 결정한다.
- `stable` 문서와 실제 API 응답 예시를 정기적으로 동기화한다.

## Later

- AI Agent 기능을 구현 범위와 데이터 계약으로 쪼갠다.
- 실증 시나리오를 UI/API/DB 요구사항으로 분해한다.
- 제어 명령, ACK/NACK, audit log 계약을 확정한다.
- Browser MCP 또는 다른 자동화 수단으로 UI 회귀 스크린샷 검증을 안정화한다.
